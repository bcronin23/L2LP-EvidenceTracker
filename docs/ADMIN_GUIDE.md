## Organisation Setup

When you first sign in, you'll be prompted to either:
- **Create a new organisation** (for your school)
- **Join an existing organisation** using an invite code

### Creating Your School Organisation

1. Navigate to the app after signing in
2. Enter your school name
3. You'll automatically become the **Admin** of this organisation

### Roles

- **Admin**: Can manage all settings, invite staff, delete evidence, and configure branding
- **Staff**: Can add students, upload evidence, and view all student data

## School Branding

Admins can customise the app appearance for their school:

### Accessing Branding Settings

1. Click **School Admin** in the sidebar (desktop) or menu (mobile)
2. Navigate to the branding section

### Customisation Options

#### Display Name
- Set a custom name to show in the app header
- Useful for showing a shorter name or nickname
- Leave blank to use your full school name

#### School Logo
- Upload your school crest or logo
- Supported formats: PNG, JPG, SVG
- Maximum file size: 5MB
- Recommended: Square image (1:1 ratio) for best display
- Logo appears in the header on both desktop and mobile

#### Accent Colour
- Choose from 8 preset colours that match your school colours:
  - Emerald (default)
  - Blue
  - Violet
  - Rose
  - Amber
  - Teal
  - Indigo
  - Slate

## Staff Management

### Generating Invite Codes

1. Go to **School Admin** > **Staff Management**
2. Click **Generate Invite Code**
3. Share the 8-character code with your staff
4. Codes expire after 7 days

### Managing Staff

- View all current staff members
- Promote staff to admin role
- Remove staff from your organisation

### Staff Onboarding

When staff join with an invite code:
1. They create an account or sign in
2. They enter the invite code
3. They're added to your school as **Staff**

## Evidence Storage

### How Evidence Files Are Stored

- All uploaded evidence (photos, videos, documents) is stored securely in private cloud storage
- Files are organised by organisation and student
- Access is controlled through temporary signed URLs (1-hour validity)

### Privacy & Security

- Evidence files are **private by default**
- Only authenticated staff from your organisation can access files
- Files are never publicly accessible via direct URLs
- Signed URLs expire after 1 hour and cannot be shared externally

### Deleting Evidence

- **Only Admins** can delete evidence
- Evidence is soft-deleted (can be recovered if needed)
- Staff cannot permanently remove evidence

### Supported File Types

| Type | Formats | Max Size |
|------|---------|----------|
| Photos | JPG, PNG, GIF, HEIC | 50MB |
| Videos | MP4, MOV, WEBM | 500MB |
| Documents | PDF, DOC, DOCX | 25MB |
| Audio | MP3, WAV, M4A | 50MB |

## Frequently Asked Questions

### Can staff delete evidence?
No, only Admins can delete evidence to maintain data integrity.

### How long do invite codes last?
Invite codes expire after 7 days. Generate a new code if needed.

### Is student data shared between schools?
No, all data is completely isolated per organisation. Staff can only see data from their own school.

### Can I change my school's primary admin?
Contact support to transfer admin ownership.

### How do I export student data?
Export functionality is planned for a future update. For now, use the print function in your browser for individual student reports.
